### 2. Customize Content
Edit these files to personalize:

**Hero Section** (`src/sections/Hero.jsx`):
- Update your introduction text

**About Section** (`src/sections/About.jsx`):
- Update your bio
- Adjust skill levels (currently at 90%, 85%, 95%)

**Projects Section** (`src/sections/Projects.jsx`):
- Update project URLs
- Add more projects or remove existing ones
- Update descriptions

**Experience Section** (`src/sections/Experience.jsx`):
- Update roles and dates
- Add more experiences

**Footer Section** (`src/sections/Footer.jsx`):
- Update social media links
- Update email address

### 3. Run the Website

```bash
# Install dependencies (if not already done)
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
